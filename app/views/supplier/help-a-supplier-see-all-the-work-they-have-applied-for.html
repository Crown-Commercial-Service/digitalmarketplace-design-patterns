Help a supplier see all the work they’ve applied for

This design allows suppliers to view all the work they’ve applied, or are currently applying, for.

How the design works

When suppliers sign into their accounts, they can view a list of all the contracts they’ve applied for by framework. Only current framework iterations are shown.

Suppliers can view by:

  • applications started
  • applications made

Each entry in the list displays the supplier name, deadline and application status, for example won or unsuccessful.

If the work has been awarded, suppliers can see:

  • who won the contract
  • the size of the winning company
  • how much the contract was worth
  • when work started on the contract

Why this is useful

This design gives suppliers a record of all the work they’ve applied for. It also allows them to see who won the contract if their application was unsuccessful.

Things to consider

It’s worth considering:

  • how to refer to work applied for, the Digital Marketplace uses the term ‘opportunities’ but we don’t know how universally understood this is by suppliers and buyers
  • how to display the work applied for, at the moment it’s just one long page
  • adding an expired status for old framework iterations
  • making the link to the work applied for easier to find
