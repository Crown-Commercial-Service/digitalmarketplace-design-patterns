Help a supplier ask a question about a framework

This design helps a supplier asks questions about a framework and see what others have asked.

How the design works

Use this design if you need to:

  • help users ask questions
  • make sure the same information is available for all suppliers at the same time

Suppliers can submit a question about their application, or the framework, through their account. Suppliers can find this function under ‘Communications’ on their account page.

Once submitted, the answer will be published publicly along with all other questions and answers for that week.

For framework clarification questions, responses are split into 3 separate PDFs containing:

  • legal questions
  • commercial questions
  • questions about their application

Any information that could be sensitive or used to identify suppliers is removed before publication.

Why this is useful

This provides a way for suppliers to ask questions and resolve any problems before the framework closes for applications.

This also means all answers are available to all suppliers at the same time. So no suppliers have information that could put them in a favourable position.

Things to consider

It might be worth considering that:

  • suppliers often repeat questions that have already been asked which suggests they are not seeing the answer to the original question
  • the answers are published weekly in 3 separate PDFs which creates a lot of work for users
  • the distinction between the 3 PDFs isn’t clear and reflects internal departmental divisions
  • the ‘ask a question’ feature is hidden far down the page under ‘Communications which could mean users miss it entirely
  • ‘Communications’ is a rather vague title which may not always be clear to users
  • if a question hasn’t been answered in time for the weekly update, the holding text says ‘answer coming soon’ which could prompt suppliers to send the question again
  • suppliers have no way of knowing their question has been answered other than checking through all 3 PDFs
